# Gil Neto - Personal Landing Page

Portfolio website built with Astro, React, TypeScript, Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install

```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/` folder.

## Deploy to Production

```bash
sudo ./deploy.sh
```

## Tech Stack
- Astro - Static site generator
- React - UI components
- TypeScript - Type safety
- Tailwind CSS - Styling
- Framer Motion - Animations
